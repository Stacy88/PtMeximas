How to Play:

This is a simple toy - a hybrid of poker and Tetris. In this interesting implementation of Tetris shapes consist of cards, gathered together in a row, they form a 7-card combination. The stronger poker hand, the more points accrued. If the combination contains the same card is assigned a penalty.
Control Keys: W / A / S / D or
up / left / right / down, and enter.
